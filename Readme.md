# 🌤️ Weather App

Welcome to my weather app project! This app allows you to check the current weather conditions for any city in the world. Simply enter the name of the city and get real-time weather data instantly!

# 🚀 Getting Started

To run the app, simply open the index.html file in a web browser. Enter the name of a city in the input field and click on the "Get Weather" button to display the weather information.

# 🛠️ Built With

HTML5
CSS3
JavaScript
Weatherbit API
